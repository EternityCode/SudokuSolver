Sudoku Solving Algorithm Demonstration – Vincent Chung

Notice: Please use the JavaScript console to call functions and manipulate the Sudoku matrix.

Defined Variables and Functions

  • sudoku (namespace)
    • .grid – Two-dimensional 9x9 Sudoku array holding integers of [1,9]
    • .solve() – Solves the current sudoku.grid
    • .insert_valid(row, col, num) – Checks that putting a num into [row,col] is a valid operation
    • .find_unassigned() – Returns the sudoku.grid coordinates of an empty/unassigned cell, or [-1,-1] if all are filled
    • .grid_valid() – Checks that every assigned cell in sudoku.grid is valid relative to its neighbours
  • sudoku_update_display() – Refreshes the web page Sudoku grid with the current sudoku.grid
  • sudoku_execute() – Attempts to solve the current sudoku.grid and update the display
